# Machine Learning Algorithms

This repository contains implementations of core supervised Machine Learning algorithms using Python.

## Algorithms Included

- Linear Regression
- Logistic Regression
- Decision Tree
- Support Vector Machine (SVM)

Each notebook demonstrates:

- Data generation / preprocessing
- Train-test splitting
- Model training
- Prediction
- Basic evaluation and visualization

## Technologies Used

- Python
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook / Google Colab

## Purpose

This project was created to strengthen understanding of fundamental Machine Learning algorithms and their practical implementation.
